[@nanaki14/match-height](npmjs.com/package/@nanaki14/match-height)

## About

要素間の高さを揃えるライブラリ

## How To Use

```js
import { matchHeight } from '@nanaki14/match-height'

const mh = matchHeight()

// 初期化
mh.init()

// addEventListener - resize
mh.watch()

// removeEventListener - resize
mh.stop()
```
